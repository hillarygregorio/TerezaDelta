## Descrição do Projeto 
 Realizei o desenvolvimento desse site como um projeto acadêmico na ETEC Lauro Gomes, com o objetivo de apresentar uma plataforma institucional inspirada na Fundação Salvador Arena. O projeto utiliza como base
o nome Tereza Delta, representando uma política brasileira e única prefeita de São Bernardo do Campo, como forma de estudo e contextualização do conteúdo. A aplicação foi criada com foco em organização das 
informações, identidade visual clara e navegação simples, simulando um site institucional moderno e funcional.
